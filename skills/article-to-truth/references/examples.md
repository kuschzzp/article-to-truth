# 示例

## 示例 1：营销稿

Before:

```text
在当今快速发展的数字时代，AI 写作工具正在赋能内容创作者，显著提升创作效率，并为未来内容生态带来无限可能。
```

命中 pattern：

- [C02] 宏大背景铺垫
- [L02] 虚浮动词
- [L03] 过度形容
- [C08] 通用乐观结尾

After:

```text
AI 写作工具更适合处理初稿阶段的重复劳动，比如起标题、列提纲、改摘要。它能省掉一部分空转时间，但发布前仍然要补真实案例、来源和作者判断。
```

## 示例 2：公众号技术文

Before:

```text
这不仅是一个代码辅助工具，更是开发者工作方式的一次重要变革。它通过智能化能力提升效率、优化体验、增强协作。
```

命中 pattern：

- [S01] 不仅而且结构
- [C01] 意义膨胀
- [S03] 三件套
- [L04] 名词化堆叠

After:

```text
它能帮开发者写样板代码、补测试草稿、解释陌生函数。真正难的部分还在人工判断：这段代码该不该写，边界条件有没有漏，团队能不能维护。
```

## 示例 3：评测输出

```text
AI 味评分：68/100

命中 pattern：
1. [C02] 宏大背景铺垫：证据句“在数字化浪潮下……”
2. [L02] 虚浮动词：证据句“赋能企业实现增长……”
3. [S03] 三件套：证据句“提升效率、优化体验、创造价值……”

优先修改：
1. 删除开头宏大背景，直接写具体业务问题。
2. 把“赋能、增长”换成工具具体做了什么。
3. 增加适用边界，说明什么团队适合用。
```

## 示例 4：原创生成触发

User:

```text
帮我写一篇关于夕阳的散文诗
```

应触发本技能，读取 `generation.md` 和 `patterns.md`。生成时重点避免：

- [C01] 把夕阳写成宏大人生意义。
- [C08] 用“珍惜当下、未来可期”强行升华。
- [L03] 连续堆叠华丽形容词。
- [S05] 写成海报式伪金句。

处理方向：

```text
从具体画面进入，比如墙面、窗沿、尘埃、人的动作；保留留白，不解释夕阳象征什么。
```

User:

```text
我想写一篇穿越小说
```

应触发本技能，读取 `generation.md` 和 `patterns.md`。生成时重点避免：

- [C02] “在一个神秘而古老的时代”式宏大背景。
- [C01] 主角开场就承担世界命运。
- [S03] 身世、系统、使命三件套堆设定。
- [F02] 用过多小标题拆剧情。

处理方向：

```text
让人物先遇到一个具体麻烦，用动作和场景带出穿越事实，而不是先解释世界观。
```
